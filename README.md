# 🔐 Smart CLI Password Generator (Python)

A powerful command-line password generator written in Python that creates strong, random, and memorable passwords.

Unlike basic generators, this tool lets users include their own word or phrase and intelligently transforms it into a secure password using random capitalization, character substitutions, and optional symbols, numbers, and letters.

Perfect for students, developers, and anyone who wants both **security + memorability**.

---

## ✨ Features

- Custom password length  
- Choose uppercase letters  
- Choose lowercase letters  
- Choose numbers  
- Choose symbols  
- Add your own memorable word or phrase  
- Smart character substitutions (`a → @`, `e → 3`, etc.)  
- Fully randomised output  
- Input validation included  
- Uses only Python standard libraries  

---

## 🖥️ Example Run
- Enter password length: 10
- Enter a word/text to transform into a memorable password: mayank
- Include uppercase letters? (y/n): y
- Include lowercase letters? (y/n): y
- Include numbers? (y/n): y
- Include symbols? (y/n): y

## Output:
- M@y4Nk#29
